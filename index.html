Global Analysis of a Dynamical Model
for Tuberculosis Transmission

with a General Contact Function


Samuel Bowong, Jean Jules Tewa

Potsdam Institute for Climate Impact Research, Germany

Motivation & Contact Function

  • TB is a leading cause of death from a single infectious agent
  • One-third of the world's population is infected with M. tuberculosis
  • 2–3 million deaths annually; HIV, MDR-TB, poverty accelerate transmission

Contact function \(\beta(N)\):

  • Mass action: \(\beta(N) = \beta\)
  • Standard incidence: \(\beta(N) = \beta/N\)
  • Saturating: \(\displaystyle \beta(N) = \frac{bN}{1+bN+\sqrt{1+2bN}}\)
  • Assumptions: \(\beta(N)>0,\ \beta'(N)\le 0,\ (N\beta(N))'\ge 0\)

SEI Model Structure

\[ \begin{cases} \dot S = \Lambda - \beta(N) S I - \mu S,\\[4pt] \dot E = \beta(N)(1-p)SI + r_2 I - [\mu+k(1-r_1)]E,\\[4pt] \dot I = \beta(N)p SI + k(1-r_1)E - (\mu+d+\delta+r_2)I \end{cases} \]

  • \(S\) – Susceptible, \(E\) – Exposed (latent), \(I\) – Infectious
  • \(\Lambda\) – recruitment, \(\mu\) – natural death, \(p\) – fast progression
  • \(k\) – reactivation, \(r_1\) – chemoprophylaxis, \(r_2\) – treatment
  • \(d\) – disease-induced death, \(\delta\) – emigration

Basic Reproduction Number \(\mathcal{R}_0\)

\[ \mathcal{R}_0 = \frac{\beta(S_0)S_0\bigl[\mu p + k(1-r_1)\bigr]}{(\mu+d+\delta)\bigl[\mu+k(1-r_1)\bigr] + \mu r_2},\qquad S_0 = \frac{\Lambda}{\mu} \]


  • \(\mathcal{R}_0\) = average number of secondary infections
  • \(\mathcal{R}_0 < 1\) → Disease-free equilibrium (DFE) is LAS
  • \(\mathcal{R}_0 > 1\) → DFE unstable, endemic equilibrium appears

Global Stability of DFE

Theorem: If \(\mathcal{R}_0 \le 1\), the DFE \(P_0 = (S_0,0,0)\) is globally asymptotically stable.


Lyapunov function: \(V(E,I) = k(1-r_1)E + [\mu + k(1-r_1)]I\)

Time derivative: \(\dot V \le [r_2\mu + (\mu+k(1-r_1))(\mu+d+\delta)](\mathcal{R}_0 - 1)I \le 0\)


By LaSalle's invariance principle, all trajectories converge to \(P_0\).

Endemic Equilibrium

When \(\mathcal{R}_0 > 1\), there exists a unique positive endemic equilibrium \(P^* = (S^*,E^*,I^*)\).


Global stability under conditions: \(\displaystyle \frac{S}{S^*} \le \frac{E}{E^*},\; \frac{S}{S^*} \le \frac{I}{I^*}\)


Lyapunov function: \(U = S - S^*\ln S + A(E - E^*\ln E) + B(I - I^*\ln I)\) with \(A,B>0\).

Numerical Results & Conclusion

  • Case 1: \(\beta(N)=\beta\) (mass action) → threshold behavior
  • Case 2: \(\beta(N)=\beta/N\) (standard incidence) → same behavior
  • Case 3: Saturating contact function → same qualitative behavior

Conclusion: Threshold dynamics determined by \(\mathcal{R}_0\); global stability proven via Lyapunov functions; numerical simulations confirm theory.
